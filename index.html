Селектори

A Як селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування такі як: колір, фон, розмір.

#id Іентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього чере скрипти.

.classКласи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково або повністю схожі елементи оформлення. У коді HTML ім'я класу визначається через атрибут class.

.class.class2До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. у цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу.

*Іноді потрібно встановити одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт або нареслення тексту. У цьому випадку допоможе універсальний селектор, який відповідає будь-якому елементу веб-сторінки.

A BПід час створення веб-сторінки часто доводиться вкладати одні елементи всередині інших. Щоб стилі цих елементів використовувалися коректно, допоможуть вкладені селектори. Наприклад, встановити стиль для тегу b, тільки коли він розташовуєтьс всредині контейнера p. Таким чином можна одночасно встановити стиль для окремого елемента, а також елемента який знаходиться всередині іншого.

A>B Дочірнім називається елемент, який безпосередньо розташовується всередині батьківького елемента. Який елемент виступає батьківським, а який його нащадком легко з'ясувати за допомогою дерева елементів - структури відносин елементів документа між собою.

A + B Cусідні називаються елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа.

A ~ BСпоріднені селектори за своєю поведінкою схожі на сусідні селектори, але на вдміну від них стильові правила застосовуються до всіх прилеглих елементів.

[attr] Встановлює стиль для елемента, якщо задано специфічний атрибут. Його значення в цьому випадку неважливе.

[attr='value'] Встановлює стиль для елемента у тому випадку, якщо задане певне значення специфічного атрибута.

[attr^='value'] Встановлює стиль для елемента в тому випадку, якщо значення атрибута починається із зазначеного тексту.

[attr$='value'] Встановлює стиль для елемента в тому випадку, якщо значення атрибута закінчується зазначеним текстом.

[attr*='value'] Можливі варіанти, коли стиль слід застосовувати до тега з певним атрибутом, коли частиною його значення є деякий текст. При цьому точно невідомо, в якому місці зазначення включено цей текст - на початку, всередині або наприкінці. У подібному випадку слід використовувати конструкцію *=. Вона визначає, що значення атрибута містить зазначений текст.

[attr~='value'] Деякі значення атрибутів можуть перераховуватись через пробіл, наприклад імена класів. Щоб задати стиль за наявності в списку необхідного значення, потрібно використовувати конструкцію ~=.

[attr|='value'] В іменах ідентифікаторів і класів дозволено використовувати символ дефіса (-), що дає змогу створювати значущі значення атрибутів id i class.
