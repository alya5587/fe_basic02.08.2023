Cписок команд

  • git init- створити локальний репозиторій
  • git add .-додати зміст робочого каталогу для наступного коміта
  • git status - переглянути зміни в репозиторії
  • git commit -m "назва коміту/інформативний текст"-створення фіксації, яка схожа на знімок репозиторія
  • git push-відправити зміни з локального комп'ютера на сервер
  • git remote v- створення, перегляд, видалення підключень до інших репозиторіїв
  • git push origin HEAD -вказуємо в який з репозиторіїв віправляємо зміни в поточну гілку
